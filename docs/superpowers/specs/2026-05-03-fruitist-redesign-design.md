# Редизайн ritualb2b.ru — вдохновение Fruitist

**Дата:** 2026-05-03  
**Статус:** Согласовано, готово к реализации

---

## Контекст

Текущий сайт ritualb2b.ru работает функционально, но визуально перегружен: стекломорфизм, градиенты, много текстовых блоков между Hero и каталогом. Владелец посмотрел fruitist.com и хочет взять оттуда: воздух, крупные фото, минимализм, параллакс. Функционал (заявки, авторизация, личный кабинет) остаётся полностью — меняется только визуальный слой.

---

## Что меняется

### 1. Структура страницы

**Было:** Nav → Hero (текст + кнопки) → категории-тайлы → каталог → контакты → FAQ → футер

**Станет:** Nav (sticky) → Hero (full-screen фото) → Карусель фото → Каталог 2 колонки (параллакс) → Контакты → Футер

Правило: **никакого текста между Hero и каталогом.**

---

### 2. Hero-секция

- Фото товара на всю ширину и высоту экрана (`100vw × 100vh`)
- Поверх фото — текущий текст без изменений:
  - Eyebrow: «Авторское производство · Симферополь · Оптовый прайс для B2B»
  - H1: «Ритуальная мастерская»
  - Sub: «Венки · Ритуальные корзинки · Искусственные цветы · Опт»
- Тёмный оверлей (`rgba(0,0,0,0.35)`) поверх фото для читаемости текста
- Кнопка «Смотреть каталог» — якорь вниз на секцию каталога
- Фото для Hero: выбрать из Google Drive после скачивания

---

### 3. Карусель фотографий

- Располагается сразу после Hero, перед каталогом
- Бесконечная горизонтальная лента (`overflow: hidden` + CSS animation `translateX`)
- Фото едут справа налево автоматически (~30s полный цикл)
- Высота полосы: `40vh` на десктопе, `30vh` на мобиле
- Фото без текста, без подписей — только изображения
- Количество фото: 6–10 штук из Google Drive (лучшие снимки венков и корзинок)
- На hover лента останавливается (`animation-play-state: paused`)
- Фото продублированы для плавного зацикливания

---

### 4. Каталог — 2 колонки с параллаксом

**Сетка:**
- Десктоп: 2 равные колонки, `gap: 16px`
- Мобиль: 2 колонки (сохраняется), карточки чуть меньше
- Карточки появляются при скролле: `IntersectionObserver` → fade-in + slide-up

**Разделители категорий:**
- Между «Венки» и «Корзинки» — полноширинный заголовок-разделитель
- Заголовок приклеивается к верху при скролле внутри своей секции (`position: sticky`)

**Параллакс:**
- Фото внутри карточки смещаются медленнее фона при скролле (`transform: translateY` через `scroll` event или CSS `background-attachment: fixed` для section-баннеров)
- Лёгкий эффект — не резкий, subtle

---

### 5. Карточка товара — новый стиль

**Было:** Стекломорфизм, backdrop-blur, градиент, двойные тени

**Станет (вариант B):**
```
┌─────────────────────┐
│                     │
│    ФОТО ТОВАРА      │  ← aspect-ratio: 4/3, object-fit: cover
│                     │
├─────────────────────┤
│ Венок «Элегия»      │  ← font-weight: 600, тёмный текст
│ Размер M            │  ← мелкий серый текст
│                     │
│ 2 500 ₽  [+ Заявка] │  ← цена оливковая, кнопка оливковая
└─────────────────────┘
```

- Фон карточки: белый `#ffffff`
- Тень: `0 2px 12px rgba(0,0,0,0.08)` — мягкая, не кричащая
- Border-radius: `12px`
- Hover: лёгкий подъём `translateY(-4px)` + тень чуть сильнее
- Кнопка «+ В заявку»: оливковый фон `#6D8A6D`, белый текст

---

### 6. Навигация — sticky

**Было:** Фиксированная шапка с backdrop-blur

**Станет:**
- При загрузке страницы: прозрачная, текст белый (поверх Hero-фото)
- После скролла на 80px: белый фон `#ffffff`, тёмный текст — плавный переход `0.3s`
- Логотип, слоган, телефоны — остаются
- Кнопка «Заявка» — всегда видна, оливковая, справа

---

### 7. Цветовая палитра — без изменений

Текущая оливковая гамма понравилась, оставляем:
```css
--amber:   #6D8A6D;   /* основной оливковый */
--amber-d: #5A7A5A;   /* тёмнее */
--bg:      #F3F1EE;   /* тёплый молочный фон */
--text:    #1C1A18;
--text2:   #5C5850;
--text3:   #9A9590;
```

Главное изменение: **убрать backdrop-blur и glassmorphism** у карточек товаров. Компонент `.g` (glass base) заменяется на чистые белые блоки.

---

### 8. Анимации при скролле

- Каждая карточка каталога появляется при входе в viewport: `opacity: 0 → 1`, `translateY(20px → 0)`, `duration: 0.4s`
- Реализация: `IntersectionObserver` с классом `.visible`
- Карточки появляются с небольшой задержкой (`stagger: 80ms`) для волнового эффекта
- Секции Hero и карусель не анимируются (они всегда видны)

---

### 10. Фотоматериалы

- **Hero-фото:** 1 горизонтальное фото (~1920×1080), лучший снимок мастерской или товара
- **Карусель:** 6–10 фото в высоком разрешении, соотношение ~4:3 или 16:9
- **Карточки:** по 1 фото на товар, желательно на нейтральном/белом фоне
- **Источник:** Google Drive (дать доступ «Всем по ссылке» → скачаем автоматически)

- Никаких текстовых блоков-описаний между Hero и каталогом
- Никакого стекломорфизма в карточках
- Никаких новых функций — только редизайн
- Никаких изменений в PHP/API
